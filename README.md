# Smart-Attendance-System
Smart Attendance System is developed using Python that automatically records attendance using face recognition technology. It captures live video through a camera, identifies registered individuals accurately, and marks attendance without manual intervention. Attendance data is stored in excel for easy access, analysis, and reporting.
